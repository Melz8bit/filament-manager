## Work Done

### Bug Fix: White bar above nav (BOM + HTMX)
- Diagnosed white bar appearing above the nav on all pages except Account
- Raw HTML source was correct; browser DOM showed head elements (meta, link, style, script) displaced into `<body>`
- Root cause: `base.html` was saved with a UTF-8 BOM (`﻿`), which was emitted at the start of every HTML response
- Fix 1: Removed BOM from `templates/base.html` using PowerShell byte-level rewrite
- Fix 2: Added `<meta name="htmx-config" content='{"historyCacheSize": 0}'>` to prevent HTMX history snapshot/restore from causing similar DOM corruption
